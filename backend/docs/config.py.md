# Файл: `backend/config.py`

## Что это
Этот файл хранит основные настройки backend-приложения.

## Простыми словами
Когда сервер запускается, он берет отсюда:
- секретный ключ для токенов,
- время жизни токена,
- путь к базе данных,
- путь для загрузки файлов.

## Что внутри
- `BASE_DIR` — папка `backend`.
- `Config.SECRET_KEY` — ключ для подписи JWT.
- `Config.JWT_ALGORITHM` — алгоритм JWT (`HS256`).
- `Config.JWT_EXPIRES_IN` — токен живет 12 часов.
- `Config.SQLALCHEMY_DATABASE_URI` — ссылка на `backend/database.db`.
- `Config.UPLOAD_FOLDER` — базовая папка загружаемых книг.

## Что обычно меняют
- `SECRET_KEY` через переменные окружения на production.
- Путь к БД (если нужна другая база).
