# Файл: `backend/app/utils/reset_db.py`

## Что это
CLI-скрипт для полного сброса и восстановления базы данных.

## Простыми словами
Удаляет текущий `database.db` и поднимает БД в одном из режимов.

## Режимы
- `--mode full` — восстановление из `backend/sql/full_backup.sql`.
- `--mode schema` — только структура таблиц из `backend/sql/schema.sql`.
- `--mode demo` — чистая БД + демо-данные (`pcs-1-23`) через сидер.

## Когда использовать
- после случайного удаления/повреждения БД,
- для старта с чистого тестового состояния.
