# Файл: `backend/app/utils/seed_pcs_demo.py`

## Что это
Сценарий автозаполнения демо-данными.

## Простыми словами
Один запуск этого файла создает рабочий учебный набор:
- группу `pcs-1-23`,
- преподавателей,
- студентов,
- привязки преподавателей,
- расписание,
- тесты и вопросы.

## Что делает
- создает/обновляет группу;
- создает преподавателей и профили;
- привязывает предметы к группе;
- генерирует студентов с логинами вида `pcs-1-23-0001`;
- ставит пароль студенту как 4-значный код из логина;
- пересобирает расписание;
- создает тесты по предметам и активирует их.

## Для чего
Чтобы быстро получить готовую БД для проверки интерфейса и API без ручного заполнения.
