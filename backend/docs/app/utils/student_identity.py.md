# Файл: `backend/app/utils/student_identity.py`

## Что это
Утилита для надежного поиска студента по разным типам идентификаторов.

## Простыми словами
Пользователь может передать:
- полный логин,
- ID,
- номер в конце логина.

Файл пытается корректно найти одного конкретного студента и сообщает причину, если это невозможно.

## Основные функции
- `extract_student_code(login)` — берет цифровой хвост из логина (например `pcs-1-23-0017` → `0017`).
- `resolve_student_user(identifier, preferred_group)` — возвращает студента или ошибку (`missing`, `not_found`, `ambiguous`).

## Зачем это нужно
Чтобы одинаково и безопасно работали маршруты оценок, посещаемости, тестов и профиля.
