# Файл: `backend/app/routes/users.py`

## Что это
Маршруты управления пользователями и личным профилем.

## Эндпоинты
- `GET /users/avatar/<filename>` — отдать файл аватара.
- `POST /users` — создать пользователя (только админ).
- `GET /users` — список пользователей (только админ).
- `DELETE /users/<id>` — удалить пользователя (только админ).
- `GET /users/me` — текущий пользователь.
- `GET /users/me/details` — расширенные данные профиля.
- `PUT /users/me/details` — редактировать свои данные (teacher/student).
- `POST /users/avatar` — загрузить аватар.
- `POST /users/change-password` — сменить пароль.

## Простыми словами
Этот файл отвечает за «личный кабинет» и админский CRUD пользователей.

## Ограничения
- Роли строго проверяются.
- Для аватара разрешены только изображения (`jpg/jpeg/png/webp`).
- Новый пароль минимум 6 символов.
