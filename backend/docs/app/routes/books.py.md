# Файл: `backend/app/routes/books.py`

## Что это
Библиотека: локальные файлы + каталог Open Library + персональная полка.

## Эндпоинты
- `POST /books` — добавить локальную книгу (scheduler/admin).
- `GET /books` — список локальных книг.
- `GET /books/<book_id>/file` — чтение/скачивание локального файла.
- `GET /books/catalog` — каталог Open Library (поиск, фильтры).
- `GET /books/shelf` — личная полка пользователя.
- `PUT /books/shelf` — обновить элемент полки (избранное/закладка/прогресс).
- `DELETE /books/shelf` — удалить из полки.

## Простыми словами
- Локальные книги — то, что загружено в систему.
- Каталог — внешние книги из Open Library.
- Полка — персональные отметки пользователя (что читать, что прочитано, где остановился).

## Поддерживаемые форматы
`pdf, epub, txt, html, md, doc, docx, fb2, rtf`.
