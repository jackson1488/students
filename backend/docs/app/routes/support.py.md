# Файл: `backend/app/routes/support.py`

## Что это
Чат технической поддержки между пользователями и админом.

## Эндпоинты
- `GET /support/media/<filename>` — файл поддержки.
- `POST /support/send` — отправить сообщение в поддержку.
- `GET /support/contacts` — список контактов поддержки.
- `GET /support/messages` — история сообщений.

## Простыми словами
- Для обычного пользователя поддержка — это диалог с админом.
- Для админа — список обращений от всех пользователей.
- Можно отправлять текст и вложения.
