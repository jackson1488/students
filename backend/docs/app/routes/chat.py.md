# Файл: `backend/app/routes/chat.py`

## Что это
Личные и групповые чаты (student/teacher/admin мониторинг).

## Эндпоинты
### Личный чат
- `POST /chat/send` — отправить личное сообщение.
- `GET /chat/messages` — получить сообщения.
- `DELETE /chat/messages/<message_id>` — удалить сообщение у себя.
- `POST /chat/messages/clear` — очистить диалог у себя.

### Контакты и медиа
- `GET /chat/contacts` — список собеседников.
- `GET /chat/media/<filename>` — файл из чата.

### Групповой чат
- `GET /chat/group/rooms` — доступные комнаты (по группе/предмету).
- `GET /chat/group/messages` — сообщения комнаты.
- `POST /chat/group/send` — отправить сообщение в комнату.

## Простыми словами
- Личный чат разрешен только между студентом и преподавателем.
- Есть групповые комнаты: общая и по предметам.
- Удаление сообщения работает «только для себя».
- Admin может просматривать историю для контроля.
