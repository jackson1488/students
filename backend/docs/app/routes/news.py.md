# Файл: `backend/app/routes/news.py`

## Что это
Новости и замены расписания с таргетингом по группам.

## Эндпоинты
- `GET /news/image/<filename>` — картинка новости.
- `POST /news` — создать новость/замену (scheduler/rector).
- `PUT /news/<news_id>` — редактировать.
- `PATCH /news/<news_id>/status` — активировать/деактивировать.
- `GET /news` — список новостей.

## Простыми словами
Можно создавать обычные новости и «замены» (schedule_replacement),
привязывать их к группам и делать видимыми только нужной аудитории.

## Особенности
- Если картинки нет, подставляется default-изображение.
- Можно загрузить файл картинки или дать URL.
- Студенты/преподаватели видят только релевантные им новости.
