# Файл: `backend/app/routes/grades.py`

## Что это
Маршруты оценок и электронного журнала.

## Эндпоинты
- `POST /grades` — добавить оценку студенту (teacher).
- `GET /grades/journal` — журнал оценок по группе/предмету.
- `POST /grades/journal` — поставить оценку через интерфейс журнала.
- `GET /grades/<student_ref>` — получить оценки студента.

## Простыми словами
- Преподаватель ставит оценки только своим студентам и только по своим предметам.
- Журнал строится по датам с учетом расписания.
- Допустимые значения оценки: `1..5` и `НБ`.

## Безопасность
- Студент видит только свои оценки.
- Teacher/Admin видят больше, но с ролевыми ограничениями.
