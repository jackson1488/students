# Файл: `backend/app/routes/exports.py`

## Что это
Экспорт данных в `HTML`, `PDF`, `XLSX` с учетом языка интерфейса.

## Эндпоинты
- `GET /exports/grades/<student_ref>` — экспорт оценок студента.
- `GET /exports/schedule/<group_id>` — экспорт расписания группы.
- `GET /exports/homework/<homework_id>` — экспорт конкретного ДЗ.

## Простыми словами
Этот файл превращает данные из базы в готовый документ,
который можно открыть, отправить или распечатать.

## Особенности
- Поддержка языков: `ru`, `en`, `kg`.
- Локализация дней недели и подписей колонок.
- Для PDF и Excel есть отдельные генераторы оформления.
