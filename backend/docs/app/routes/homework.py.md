# Файл: `backend/app/routes/homework.py`

## Что это
Полный цикл домашних заданий: выдача, сдача, проверка, архив.

## Эндпоинты
- `GET /homework/media/<filename>` — файл вложения по ДЗ.
- `GET /homework/targets` — доступные предметы/группы/студенты для учителя.
- `POST /homework` — создать ДЗ.
- `GET /homework/<group_id>` — список ДЗ по группе.
- `POST /homework/<homework_id>/submit` — сдать ДЗ.
- `GET /homework/<homework_id>/submissions` — список сдач.
- `PATCH /homework/submissions/<submission_id>/review` — проверка сдачи учителем.
- `PATCH /homework/<homework_id>/status` — активировать/архивировать само ДЗ.
- `DELETE /homework/<homework_id>` — удалить ДЗ (admin).

## Простыми словами
- Учитель создает задание группе, всем своим группам или конкретному студенту.
- Студент отправляет комментарий/файл.
- Учитель проверяет, ставит статус и оценку.
- При необходимости ДЗ и сдачи архивируются.

## Особенности
- Есть ограничения на типы файлов.
- Ссылка на вложение становится публичным URL.
- При оценивании может создаваться запись в таблице `grades`.
