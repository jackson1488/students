# Файл: `backend/app/routes/academy.py`

## Что это
Админская учебная структура: группы, преподаватели, студенты, привязки.

## Эндпоинты
### Группы
- `POST /groups` — создать группу.
- `PUT /groups/<group_id>` — изменить группу.
- `GET /groups` — список групп.

### Преподаватели
- `POST /teachers` — создать преподавателя.
- `GET /teachers` — список преподавателей.
- `GET /teachers/<teacher_id>` — карточка преподавателя.
- `PUT /teachers/<teacher_id>` — изменить преподавателя.

### Студенты
- `POST /students` — создать студента.
- `GET /students` — список студентов.
- `GET /students/<student_id>` — карточка студента.
- `PUT /students/<student_id>` — изменить студента.

### Привязки
- `POST /bindings` — привязать преподавателя к группе и предмету.
- `GET /bindings` — список привязок.

## Простыми словами
- Админ сначала создает группу.
- Потом создает преподавателей и студентов.
- Затем делает привязки, чтобы было понятно, кто какой предмет ведет в какой группе.

## Важные детали
- Логин студента генерируется автоматически (`группа-XXXX`).
- Пароль нового студента = этот 4-значный код.
- Дата рождения валидируется в формате `YYYY-MM-DD`.
- Проверяется уникальность логинов.
