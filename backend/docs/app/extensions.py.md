# Файл: `backend/app/extensions.py`

## Что это
Общее место инициализации расширений Flask.

## Простыми словами
Здесь создается объект `db` (SQLAlchemy), который используется во всех моделях и маршрутах.

## Что внутри
- `db = SQLAlchemy()`

## Зачем вынесено отдельно
Чтобы один и тот же `db` можно было импортировать во всех файлах без циклических импортов.
