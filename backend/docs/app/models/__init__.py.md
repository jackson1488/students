# Файл: `backend/app/models/__init__.py`

## Что это
Центральный экспорт всех моделей базы данных.

## Простыми словами
Этот файл собирает все таблицы в одном месте, чтобы в других файлах можно было писать коротко:
```python
from app.models import User, Test, Homework
```

## Что внутри
- импорт `db`,
- импорт всех классов таблиц (`User`, `StudyGroup`, `Test`, `News` и т.д.),
- `__all__` со списком доступных имен.
