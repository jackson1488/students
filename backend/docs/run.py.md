# Файл: `backend/run.py`

## Что это
Точка запуска backend-сервера.

## Простыми словами
Этот файл создает приложение Flask и поднимает веб-сервер на `0.0.0.0:5000`.

## Что внутри
- `create_app()` — собирает все части backend.
- `app.run(...)` — запускает сервер в debug-режиме.

## Когда используется
Когда выполняете команду:
```bash
python backend/run.py
```
