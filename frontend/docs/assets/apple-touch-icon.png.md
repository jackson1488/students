# Файл: `assets/apple-touch-icon.png`

## Что это
Графический asset (изображение/иконка) для интерфейса.

## Простыми словами
Этот файл отвечает за внешний вид приложения на разных платформах:
- иконка приложения,
- favicon,
- splash-экран,
- PWA-иконки.

## Где используется
- Конфигурация Expo (`app.json`),
- web/PWA сборка,
- системные иконки в браузере и на телефоне.
