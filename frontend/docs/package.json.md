# Файл: `package.json`

## Что это
Главный манифест JavaScript-проекта.

## Простыми словами
Описывает проект, его зависимости и команды запуска.

## Что внутри
- `dependencies` — библиотеки приложения,
- `devDependencies` — инструменты разработки,
- `scripts` — команды npm.

## Примеры команд
name, version, main, start, start:local, start:local:clear, android, ios
