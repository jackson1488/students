Главная точка входа приложения React Native/Expo. Здесь подключается корневая навигация и провайдеры состояния.

Этот файл — одна часть фронтенда, которая либо:
- показывает UI,
- хранит состояние,
- общается с backend,
- или настраивает платформу.
