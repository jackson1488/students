# Файл: `app/i18n/en.json`

## Что это
Файл переводов для английского языка.

## Простыми словами
Здесь хранится словарь текстов интерфейса.
Каждый ключ = одна подпись/сообщение в приложении.

## Структура
- Всего ключей: **432**.
- Примеры ключей:
- `appTitle`
- `loading`
- `login`
- `logout`
- `loginLabel`
- `password`
- `oldPassword`
- `newPassword`
- `confirmPassword`
- `signIn`
- `signingIn`
- `invalidCredentials`

## Как используется
Компоненты вызывают перевод по ключу через i18n-контекст, и UI автоматически показывает нужный язык.
