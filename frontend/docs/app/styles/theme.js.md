# Файл: `app/styles/theme.js`

## Что это
Центральные цвета, размеры и стиль-система проекта.

## Простыми словами
Этот файл — одна часть фронтенда, которая либо:
- показывает UI,
- хранит состояние,
- общается с backend,
- или настраивает платформу.

## Что делает в проекте
- Импортов в файле: **1**.
- Основная ответственность: Центральные цвета, размеры и стиль-система проекта.

## Главные элементы кода
- `buildNavigationTheme`: отдельный шаг логики внутри этого файла.

## Как читать новичку
1. Сначала смотри импорты вверху.
2. Затем найди главный экспорт (`export default ...`).
3. После этого пройди функции сверху вниз — они обычно описывают полный сценарий экрана/компонента.
