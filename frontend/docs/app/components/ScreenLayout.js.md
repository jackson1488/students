# Файл: `app/components/ScreenLayout.js`

## Что это
Общий каркас экрана: отступы, заголовки, безопасные зоны.

## Простыми словами
Этот файл — одна часть фронтенда, которая либо:
- показывает UI,
- хранит состояние,
- общается с backend,
- или настраивает платформу.

## Что делает в проекте
- Импортов в файле: **4**.
- Основная ответственность: Общий каркас экрана: отступы, заголовки, безопасные зоны.

## Главные элементы кода
- `ScreenLayout`: отдельный шаг логики внутри этого файла.

## Как читать новичку
1. Сначала смотри импорты вверху.
2. Затем найди главный экспорт (`export default ...`).
3. После этого пройди функции сверху вниз — они обычно описывают полный сценарий экрана/компонента.
