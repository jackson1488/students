# Файл: `app/hooks/AuthContext.js`

## Что это
Глобальное состояние авторизации, токена и текущего пользователя.

## Простыми словами
Этот файл — одна часть фронтенда, которая либо:
- показывает UI,
- хранит состояние,
- общается с backend,
- или настраивает платформу.

## Что делает в проекте
- Импортов в файле: **4**.
- Основная ответственность: Глобальное состояние авторизации, токена и текущего пользователя.
