# Файл: `package-lock.json`

## Что это
Lock-файл npm с точными версиями зависимостей.

## Простыми словами
Этот файл фиксирует, какие именно версии библиотек должны установиться на любой машине.

## Зачем нужен
- стабильные сборки,
- одинаковое окружение у всех разработчиков,
- меньше «у меня работает, у тебя нет».
